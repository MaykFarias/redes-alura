# Redes Alura

Repositorio com exercicios do curso de Redes da Alura no Cisco Packet Tracer.

## Modulos
- Modulo 01 - Conceitos iniciais e criacao de uma intranet
- Modulo 02 - VLANs, politicas de acesso e conexao com internet
- Modulo 03 - Roteamento, DNS e IPv6
- Modulo 04 - Wi-Fi, criando uma rede sem fio de modo seguro
