# Packet Tracer - Intranet DHCP DNS Lab

Projeto de simulação de uma intranet corporativa utilizando Cisco Packet Tracer.

---

# 📌 Objetivo

Implementar uma rede interna com:

- DHCP
- DNS
- Servidor WEB
- Roteamento entre redes
- IP estático e dinâmico

A rede permite acessar uma página interna através do domínio:

```txt
http://intranet.com
```

---

# 🖥️ Estrutura da rede

- 1 Roteador Cisco 1841
- 3 Switches Cisco 2950
- 3 Computadores
- 2 Servidores

---

# 🌐 Configurações realizadas

## DHCP
O roteador fornece IP automaticamente para os computadores.

## DNS
O servidor DNS realiza a resolução do domínio:

```txt
intranet.com
```

## HTTP
O servidor WEB hospeda a página da intranet.
